Le code génetique

le code génetique est un système de
correspondance entre l'ordre du gène et l'ordre de la protéine. Il est redondant ou dégénéré lorsqu'il est désigné par plusieurs codons. les uns sont initiateurs c'est à dire qu'ils désignent le debut d'une synthèse et les autres sont appelés codons stop, ils marquent la fin d'une synthèse.
